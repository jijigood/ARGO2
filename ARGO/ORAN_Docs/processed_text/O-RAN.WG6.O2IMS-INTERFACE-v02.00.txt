2.1.1.1.3 ResourceType 267 
The ResourceType service object represents a resource type in an O-Cloud. 268 
Resource types provide the categorization of O-Cloud Nodes as described in the O-RAN Cloud Architecture and 269 
Deployment Scenarios [7] specification. The values for these types are used to build the inventory and enable an 270 
authorized consumer to track resource changes over time. The O-Cloud provides a mechanism to discover the resource 271 
types with respective operations on resource types. 272 
Please note that the list of resource classes is not exhaustive, and additional classes are FFS.  273 
The information representing the ResourceType shall follow the provisions indicated in table 2.1.1.1.3-1. 274 
Table 2.1.1.1.3-1: Attributes of the ResourceType 275 
Attribute Qualifier Description 
resourceTypeId M Identifier of the resource type. Globally unique across O-Cloud 
instances. 
name M Human readable name of the resource type. 
description M Human readable description of the resource type. 
vendor M Name of the provider of the resource type. 
model M Information about the model of the resource type. 
version M Version or generation of the resource type. 
alarmDictionary M Contains a dictionary of alarms that can be generated by the 
resource type, if no alarms are reported against a resource of this 
resourceType then the dictionary is not populated. 
resourceKind M The enumerated set of values representing kind (UNDEFINED,

are tailored by the vendor to extend the information provided about 
the Resource Type.

2.1.3.1 Alarm Services 665
