Chapter 2. Management Services 1
2.1 Provisioning Management Services 2 Provisioning management services allow a Provisioning MnS Consumer to configure attributes of managed objects on 3 the Provisioning MnS Provider that modify the Provisioning MnS Provider’s capabilities in its role in end-to-end 4 network services and allows a Provisioning MnS Provider to report configuration changes to the Provisioning MnS 5 Consumer. NETCONF is used for the Provisioning Management Services to Create Managed Object Instance, Delete 6 Managed Object Instance, Modify Managed Object Instance Attributes and Read Managed Object Instance Attributes. 7 A REST/HTTPS event is used to notify the Provisioning MnS subscribed Consumers when a configuration change 8 occurs.   9 Use cases are from 3GPP TS 28.531. 10 Reference documents include 3GPP TR 28.890, TS 28.531, RFC 6241, “Network Configuration Protocol 11 (NETCONF)”, IETF, RFC 7950, “The YANG 1.1 Data Modeling Language”. 12 2.1.1 General NETCONF Requirements 13 REQ-GNC-FUN-1:  The provisioning management service provider and consumer SHALL support the following 14 NETCONF operations: 15 • get 16 • get-config 17 • edit-config 18 • lock 19 • unlock 20 • close-session 21 • kill-session 22 Other operations are optional. 23 REQ-GNC-FUN-2:  The provisioning management service provider and consumer SHALL support the following 24 NETCONF capabilities: 25 • writable-running 26 • rollback-on-error 27 • validate 28 • xpath 29 Other capabilities are optional. 30 REQ-GNC-FUN-3:  The provisioning management service provider and consumer SHALL support a running datastore 31 for NETCONF.  Support for a candidate datastore is optional. 32 REQ-GNC-FUN-4:  The provisioning management service provider and consumer SHALL support YANG1.1, defined 33 in RFC 7950, including coexistence with YANG Version 1 as specified therein. 34 2.1.2 Create Managed Object Instance 35 2.1.2.1 Description 36 Provisioning MnS Consumer sends a synchronous provisioning update request to the Provisioning MnS Provider to 37 create a Managed Object Instance (MOI) on the Provisioning MnS Provider and set its attribute values.  38
